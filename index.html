Latest code examples

My latest solution on codewars for 6 kyu task called "Sort the Odd":

function sortArrayIfOdd(array) {
  const odd = array.filter(el => el % 2).sort((a, b) => a - b);
  return array.map(el => el % 2 ? odd.shift() : el)
}
